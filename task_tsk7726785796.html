Embedded cognitive trap · not shown to the model

Lazy-AI failure mode Uses the headline Rs 3,240 Cr detergent revenue as the projection base. This inflates FY30 by ~Rs 890 Cr. The decision happens to remain GO, so the error surfaces only as a governance / numeric-integrity breach, not a flipped recommendation. Expert path Read Note 7, strip the Rs 620 Cr demerged Soda Ash unit to reach the Rs 2,620 Cr continuing base, project to Rs 3,761 Cr, size SAM Rs 23,040 Cr and premium Rs 3,226 Cr, and build the IRR correctly — full capex at Year-0, tax on EBIT (not EBITDA), working capital applied, terminal value on the correct Year-5 EBITDA — landing IRR ≈ 53.9% > 18% → GO.
